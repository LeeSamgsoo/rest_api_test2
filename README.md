# 특이사항
- 게시글 생성, 수정, 삭제 할 때 로그인 상태여야 가능하게 했습니다.
- 로그인은 spring securit나 jwt를 쓰지 않고 간단하게 쿠키에 username을 저장하는 방식으로 했습니다.
